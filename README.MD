# NITKA — Handmade Knitwear (website)

A small, fast website for **NITKA**, a handmade-knitwear brand — knitted accessories
made by hand in small batches (keychains, scarves, brooches, jewelry). Single-page site,
deployed on Vercel.

## About

Active URL - 'https://nitka.click/'

- Single-page site presenting the brand and its handmade products
- Clean, lightweight front end — loads fast
- SEO-ready: meta description, Open Graph tags, `sitemap.xml`, `robots.txt`, favicon
- Deployed on **Vercel** with SPA routing (all routes rewrite to `index.html`)

## What I practiced here

- Building and shipping a real, live website end to end
- Front-end structure, assets, and responsive layout
- SEO basics and social-share previews (Open Graph)
- Deployment and routing config on Vercel
- Connect and manage domen name

## Tech

HTML · CSS · JavaScript · Vercel
